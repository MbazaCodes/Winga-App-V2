# WINGA APP PLATFORM

Tanzania's Trusted Shopping Guide Platform

## Platform Structure

```
WINGA APP PLATFORM/
├── 📱 customer-pwa/      Customer PWA (Next.js 16 + React 19)
├── 🛠️  winga-pwa/        Winga PWA (Next.js 16 + React 19)
├── 💻 admin-web-app/     Admin Web App (Next.js 16 + TypeScript)
└── 📲 mobile-app/        Mobile App (Flutter — Future Phase)
```

## Quick Start

### Customer PWA
```bash
cd customer-pwa
npm install
npm run dev        # http://localhost:3000
```

### Winga PWA
```bash
cd winga-pwa
npm install
npm run dev        # http://localhost:3000
```

### Admin Web App
```bash
cd admin-web-app
npm install
npm run dev        # http://localhost:3001
```
